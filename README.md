# カレンダー共有アプリ

シンプルで確実に動作するカレンダー共有アプリです。

## 🚀 確実なデプロイ方法

### GitHubを使った自動デプロイ（推奨）

1. **GitHub.com** で新しいリポジトリを作成
2. 以下のファイルをアップロード：
   - `index.html`
   - `styles.css` 
   - `script.js`
   - `_headers`
   - `_redirects`
3. **Cloudflare Pages** でリポジトリを連携
4. 自動デプロイ完了

## 📁 ファイル構成

```
├── index.html     # メインHTML
├── styles.css     # スタイル
├── script.js      # JavaScript
├── _headers       # CORS設定
└── _redirects     # リダイレクト設定
```
