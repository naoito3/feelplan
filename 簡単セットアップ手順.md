
## ✅ 解決策：GitHubを使った自動デプロイ

### ステップ1: 最小限のファイル構成

以下の**5つのファイルだけ**を使用：
- `index.html` ✅
- `styles.css` ✅  
- `script.js` ✅（ローカルストレージ版）
- `_headers` ✅
- `_redirects` ✅

### ステップ2: GitHubにアップロード

1. **GitHub.com** → **New repository**
2. リポジトリ名：`calendar-app`
3. **上記5ファイルのみ**をアップロード
4. **Commit changes**

### ステップ3: Cloudflare Pagesで自動デプロイ

1. **Cloudflare Dashboard** → **Pages**
2. **Connect to Git** → GitHubリポジトリを選択
3. **Build settings:**
   - Framework preset: `None`
   - Build command: （空欄）
   - Build output directory: `/`
4. **Save and Deploy**

### ステップ4: 動作確認

✅ デプロイ完了後、以下が動作することを確認：
- カレンダー表示
- イベント作成・編集・削除
- ローカル保存（ブラウザ内）
- モバイル対応
